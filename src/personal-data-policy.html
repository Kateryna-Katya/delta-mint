Политика обработки персональных данных

1. Общие положения

Настоящая политика (далее — «Политика») определяет порядок и условия обработки данных, предпринимаемые платформой Delta-Mint (далее — «Оператор»), и устанавливает передовые меры по обеспечению цифровой безопасности этих данных.

1.1. Важнейшей целью Оператора является соблюдение прав и свобод человека при внедрении AI-инструментов, включая защиту прав на неприкосновенность частной жизни и личную тайну в соответствии с регламентами ЕС.

1.2. Настоящая Политика применяется ко всей информации, которую Оператор может получить о посетителях (далее — «Пользователи») веб-сайта delta-mint.fit.

2. Основные понятия

  • Веб-сайт — совокупность графических материалов и программных решений для ЭВМ, обеспечивающих доступ к практикам применения ИИ по адресу delta-mint.fit.
  • Пользователь — любой посетитель платформы.
  • Персональные данные — любая информация, относящаяся к определенному Пользователю (имя, контактные данные).
  • Обработка данных — любое действие с использованием средств автоматизации (ИИ-алгоритмов) или без них, включая сбор, хранение и систематизацию информации.
  • Безопасность — защищенность данных от неправомерного доступа, обеспечиваемая нашей передовой цифровой инфраструктурой.

3. Данные, которые мы обрабатываем

Оператор обрабатывает следующие категории:

  • Предоставляемые Пользователем:
    • Фамилия, имя;
    • Электронная почта (Email);
    • Номер телефона (только цифры).
  • Собираемые автоматически:
    • Обезличенные данные о посетителях (файлы cookie), собираемые для аналитики методологий, доказавших эффективность.

4. Цели обработки

  • Предоставление доступа к AI-инструментам для реальных задач.
  • Установление обратной связи для экспертной поддержки на всех этапах.
  • Заключение договоров на обучение и участие в программах пассивного дохода.
  • Информирование о новых решениях, которые меняют правила игры.

5. Правовые основания

Оператор обрабатывает данные только на законных основаниях:

  • При наличии добровольного согласия Пользователя, выраженного через формы на сайте.
  • Если это разрешено в настройках вашего браузера (cookie и JavaScript включены).

6. Порядок хранения и безопасности

  • Оператор принимает все возможные технические меры для защиты данных в соответствии со стандартами Италии и ЕС.
  • Персональные данные никогда не будут переданы третьим лицам без вашего согласия, кроме случаев исполнения законодательства.
  • Срок обработки данных неограничен. Пользователь может отозвать согласие, направив уведомление на наш email.

7. Заключительные положения

Пользователь может получить любые разъяснения по вопросам обработки данных, обратившись к Оператору: hello@delta-mint.fit
